## Change: fe-editor-context-menu-and-tooltips

### Requirement: 编辑区必须提供自定义右键菜单 [ADDED]

#### Scenario: 编辑区右键弹出 Context Menu（非浏览器默认） [ADDED]

- **假设** 用户在编辑器正文区域右键点击
- **当** 右键菜单出现
- **则** 系统必须展示自定义 Context Menu（本地化文案）
- **并且** 菜单项应包含常用编辑动作与 AI 动作入口
