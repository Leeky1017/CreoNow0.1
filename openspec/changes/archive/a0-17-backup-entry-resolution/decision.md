## 三、最终决策

### 选定方案: S1 — 隐藏备份入口

### 决策理由

1. **100% 假功能无需保留入口**（A0-08 Q1-Q10）：备份链路从 UI 到后端完全为空。这不是「部分实现」，而是「完全虚假」。
2. **硬编码欺骗必须消除**（A0-08 Q8）：「上次备份：2 分钟前」为永恒不变的假时间戳，对用户构成数据安全层面的误导。
3. **Coming Soon 不适用**（A0-08 Q10 + A0-15 策略分析）：备份功能无 Spec 定义、无路线图 Issue、无用户需求信号。
4. **最小成本最大诚实**：S1 是纯减法，开发成本 S 级，风险极低，可逆性高。
